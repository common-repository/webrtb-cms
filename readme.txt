De WebRTB CMS plugin verandert uw Wordpress in een WebRTB CMS.

== Installation ==

1. Upload the entire `webrtb-cms` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'WebRTB' menu in your WordPress admin panel.
